
# 🏥 Hospital Management Data Analysis (SQL Project)

## 📌 Project Overview
This project focuses on analyzing hospital data using SQL to generate business insights related to patients, doctors, appointments, treatments, and revenue.

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Data Analysis Techniques

## 📊 Key Analysis Performed
- Patient registration trends and recent activity analysis  
- Doctor specialization and experience-based segmentation  
- Appointment status distribution and patient engagement analysis  
- Treatment cost analysis (Min, Max, Avg)  
- Billing and payment status analysis  
- Revenue generation and high-value patient identification  

## 📈 Advanced Analytics
- RFM Segmentation (Recency, Frequency, Monetary)  
- Patient segmentation (Champion, Loyal, At Risk)  
- Doctor workload analysis  
- Outlier detection in treatment cost  
- Ranking of doctors and patients using window functions  

## 🔍 Key Insights
- Identified low patient acquisition trends  
- Found high no-show appointments indicating disengagement  
- Highlighted top revenue-generating patients  
- Analyzed demand for medical specializations  

## 🗂️ Database Tables Used
- Patients  
- Doctors  
- Appointments  
- Treatments  
- Billing  

## 💡 Key Learnings
- Advanced SQL queries (JOIN, GROUP BY, HAVING)  
- Window Functions (RANK, NTILE, LAG)  
- Data-driven decision making  
- Business problem solving using SQL
